10:36 was the final project for my internet programming class. i got full creative control over how the blog looked,
as long as i had all of the required back-end/databse parts. it is a full-stack blogging website dedicated to the discussion
of music, fashion, beauty, and more. the stack consists of:

  • front-end: pug (HTML template engine), CSS, JavaScript
  • back-end: express.js (node.js web application framework)
  • database: mySQL

this project showed me how much i love creating full-stack applications and being creative.

a live demo is currently unavailable, but will be uploaded in the near future. you can view the code here.
note that this is not functional as you need my school credentials to access the database.

showcase:

homepage

features the front page of the blog, previewing the latest 5 posts

browse posts

includes a way to browse all posts, using pagination. can go to previous or following pages of posts

create post

create post form, used to make a new post using API

view post

view a post by clicking on their title or date, brings you to post page. gives option to edit post, delete post, or post and delete comments

create comment

create comment form, used to make a new comment using API
